VIM Syntax Highlighting for XASM
================================

Installation
------------

Put xasm.vim in your .vim/syntax directory.

Put these lines in your .vimrc:

    au BufRead,BufNewFile *.asx set filetype=xasm
    au BufRead,BufNewFile *.asm set filetype=xasm
